🛒 Supermarket Sales & Profit Analysis Dashboard

#### 📌 Overview
This project analyzes supermarket sales, profit, and cost across different countries, regions, and time periods. The dashboard helps in understanding business performance and identifying trends.

#### 🎯 Key Features
- KPI cards for Total Sales, Profit, and Cost
- Sales analysis by category and sub-category
- Region and country-level insights using maps
- Time-based analysis (year, quarter, month)
- Top customers and product performance
- Key Influencers visual for profit analysis

#### 🔍 Insights
- Sales and profit show steady growth over the years
- Certain categories contribute more to total revenue
- Profit is directly affected by sales and cost changes
- Some regions consistently perform better
